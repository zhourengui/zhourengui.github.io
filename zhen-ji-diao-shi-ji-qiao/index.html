真机调试技巧

Safari + IOS

首先，打开Mac电脑上的Safari浏览器，然后点击Safari浏览器>偏好设置>高级，然后将在菜单栏中显示开发菜单打开，这时候就会出现开发的tab。
然后，苹果手机用数据线连接电脑，在手机的设置中打开设置>Safari浏览器>高级>，将网页检查器打开。
然后手机使用Safari访问要测试的地址，使用Mac电脑的ip访问，这时候Mac电脑的Safari的开发tab就会出现手机的选项，点击调试即可。

Chrome + Android

在Chrome浏览器输入chrome://inspect/#devices

Android需要打开开发中模式，然后打开USB调试，用数据线连接电脑，手机浏览器访问要调试的地址，然后Chrome浏览器就会监测到设备，然后在Chrome点击inspect进行调试，如果长时间白屏的情况，让手机和电脑浏览器保存最新版本。

Fiddler / Charles

首先当然是下载Charles，然后设置Proxy Setting，设置的端口是要本机没有用到的端口，否则通过不了，然后在macOS标签将enable macOS proxy开启。然后就配置好了。然后在手机网络中配置代理，服务器输入电脑的ip地址，端口设置我们在Proxy Setting设置的端口号，然后就ok了，然后在手机访问的任何接口就能被抓取。

Weinre / Spy-Debugger / vConsole

通过添加script标签脚本来进行调试

上面的所有调试都是基于手机和电脑处于同一个局域网的情况下才能调试。
