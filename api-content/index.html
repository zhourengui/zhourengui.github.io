Monad是一种设计模式，表示将一个运算过程，通过函数拆解成互相连接的多个操作。你只要提供下一步运算所需的函数，整个运算就会自动进行下去。 Promise就是一种Monad。 Monad可以避开地狱嵌套，可以轻松地进行深度嵌套的函数式编程。 Monad的作用
