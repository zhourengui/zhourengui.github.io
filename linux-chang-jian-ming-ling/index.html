Linux 常见命令

前言：

  1. 文档型命令：文件相关命令（touch，cat，echo，rm, vi, cd）

  2. 硬件型：硬盘/进行/服务/网络

  3. 功能型：压缩/解压，下载，远程

文档型命令

  1. touch test.json 创建文件
  2. cat test.json 查看文件内容
  3. vi test.json 编辑文件内容
  4. echo
# 向文件test.json末尾添加123456
echo 123456 >> test.json
# 修改test.json的内容为123456
echo 123456 > test.json
  1. rm
#删除文件
rm test.json
#删除目录
rm -r test

硬件型

  1. lsb_release -a查看系统描述
  2. df -Th 查看磁盘命令
  3. top 查看系统cup和内存资源
  4. ps -ef | grep docker，grep代表过滤，这里表示过滤出docker的进程
  5. kill -9 pid，杀进程
  6. service sshd status，查看服务命令
  7. service sshd restart，重启sshd服务
  8. systemctl status firewalld.service，查看防火墙的服务

功能型

  1. wget https://baidu.com 下载资源
  2. tar zxvf . 解压tar文件

zxvf意思代表.gz结尾的文件x代表解压缩，v显示所有解压的过程，f代表解压缩后的名称是压缩包的前缀

  1. tar zcvf 压缩后的名称 需要压缩的目录或文件，c代表压缩，

  2. scp

文件上传到服务器

# 文件
scp 文件 root@47.98.252.43:/root/source
# 文件夹
scp -r 文件夹 root@47.98.252.43:/root/source

服务器的文件下载到本地

scp root@47.98.252.43:/root/source 本地目录
